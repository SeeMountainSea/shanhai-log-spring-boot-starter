ShanHaiLog 主要提供以下能力：

- 支持基于Spring Aop+@RequestLog注解方式记录API(请求报文/响应报文/请求时间/请求用户/请求URI等)标准化日志
- 支持Spel表达式进行动态参数获取
- 支持自定义获取当前用户信息
- 支持自定义获取文件下载与文件上传信息
- 支持自定义日志存储方式
- 支持日志标准化写入操作系统临时目录（日志存储默认实现）
